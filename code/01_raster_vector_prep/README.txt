Run in the following steps.

1_urban_rural.py - This makes an urban/rural raster from the GHS-SMOD urban/rural land use types.
2_inland_pourpoints_drop.py - This finds the inland pourpoints/watersheds and drops them.
3_river_raster.py - This makes a raster at 1-km of major rivers (Hydrosheds data) and coastlines needed for the effluent rst.
